Reading Image

Lakukan Reading Image terhadap gambar yang sudah kita siapkan sebelumnya. Disini saya menggunakan gambar Logo UIN Malang yang kemudian saya simpan kedalam dataset yang bernama img1. 

img1 <- readImage('Logo.jpg')
display(img1, method="raster")

Beri Text

Setelah melakukan Reading Image kita mulai proses Watermarking Digital. Berikan text kedalam logo tersebut lalu simpan menjadi gambar baru. Disini saya simpan gambar yang telah diberi text dengan nama “relogo.jpg” 

display(img1, method="raster")
text(x = 850, y = 810, label = "Logo", adj = c(2,2), col = "#00964A", cex = 3)
text(x = 855, y = 815, label = "Logoo", adj = c(2,2), col = "#00964A", cex = 3)
text(x = 860, y = 820, label = "Logooo", adj = c(2,2), col = "#00964A", cex = 3)

colorMode(img1) = Color
filename = "relogo.jpg"
dev.print(jpeg, filename = filename, width = dim(img1)[1], height = dim(img1)[2])

Pembuktian

Beri Tanda

Sebagai pembuktian pertama mari kita lihat lingkaran merah yang ada dibawah kanan tersebut. Didalam tanda lingkaran kanan tersebutlah letah dari tulisan watermark kita. 

img3 <- readImage('logomarked.jpg')
display(img3, method="raster")

Tampilkan Data Histogram

Selanjutnya mari kita buktikan dengan menggunakan historgram. Tampak data dari kedua histogram antara “img1” dan “img2” memiliki tinggi yang berbeda. Ini artinya intesitas warna didalam pikselnya juga berbeda 

hist(img1)

hist(img2)

Tampilkan Data Matrix Img

Untuk lebih detailnya kita bisa juga buktikan dengan menampilkan matrix untuk membandingkan numeric pixel dari gambar yang belum dan sudah diberi watermark. 

img1_data <- as.numeric(img1)
img1_data[497:505]

img2_data <- as.numeric(img2)
img2_data[497:505]
